#### ‚ö†Ô∏è Breaking Changes

<Warning>
**Migration Required**: This release includes breaking changes. Please review the migration guide below.
</Warning>

**API Endpoint Changes:**
```diff
- POST /v1/kyc/verify
+ POST /v1/kyc/sessions

- GET /v1/kyc/status/{id}
+ GET /v1/kyc/sessions/{sessionId}

- POST /v1/attestations/create
+ POST /v1/attestations
```

**Response Format Changes:**
```diff
// Old format
{
  "success": true,
  "result": { ... }
}

// New format
{
  "success": true,
  "data": { ... },
  "requestId": "req_123456789"
}
```

#### üîÑ Migration Guide

<Steps>
  <Step title="Update API Endpoints">
    Replace old endpoint URLs with new v1 paths
  </Step>
  <Step title="Update Response Handling">
    Change `result` to `data` in response parsing
  </Step>
  <Step title="Update Authentication">
    Migrate to new JWT-based authentication
  </Step>
  <Step title="Test Integration">
    Verify all functionality with the new API version
  </Step>
</Steps>

### Recent Security Patches

- **Fix**: Enhanced constant-time comparison algorithms
